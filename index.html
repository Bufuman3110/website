Comandi Utili per Windows

CMD

  • netsh wlan show profiles
  • netsh wlan show profiles “nome rete” key=clear: Per reti memorizzate, su contenuto chiave trovi la password della rete.

DRIVERQUERY

driverquery serve a fare la lista dei driver del computer con tutti i dettagli.

Comandi:

  • driverquery -v
  • driverquery -v > listadriver.txt: salverà la lista in un file di testo nella cartella corrente.

SHUTDOWN

Il comando shutdown spegne il computer, che si può fare col pulsante senza problemi.

Comando utile:

  • shutdown /r /o: Riavvia il PC e lancia il menu di opzioni avanzate di avvio per accedere alla modalità provvisoria e all'utilità di recupero.

SFC (System File Checker)

Serve a fare il controllo dei file di sistema per correggere errori di file non trovati o corrotti.

  • sfc /scannow: rileva file danneggiati o mancanti e sostituisce con file buoni.

TASKLIST

tasklist funziona come un task manager per vedere tutte le attività in esecuzione.

  • tasklist -svc: Mostra i servizi relativi a ciascuna attività.
  • tasklist -v: Fornisce maggiori dettagli su ogni processo.
  • tasklist -m: Individua i file DLL associati a processi attivi.

TASKKILL

Ogni processo che appare in tasklist può essere terminato con il comando:

  • taskkill -im nomefile
  • taskkill -pid IDprocesso

ASSOC

assoc è il comando per vedere le associazioni dei file con i programmi.

  • assoc .txt: Vedi quale programma apre i file .txt.
  • assoc .txt=: Cambia il programma associato.

FC (File Compare)

fc serve a confrontare due file di testo per vedere se ci sono differenze.

  • Esempio: fc /l "C:\Program Files\esempio1.doc" "C:\Program Files\esempio2.doc"

IPCONFIG

Mostra l'indirizzo IP del computer collegato alla rete.

  • ipconfig /release e ipconfig /renew: Chiedere un nuovo indirizzo IP.
  • ipconfig /flushdns: Aggiorna la risoluzione dei nomi DNS.
  • ipconfig /all: Mostra tutti gli indirizzi di rete del computer.

NETSTAT

Il comando netstat -an mostra la lista delle porte di rete aperte e relativi IP.

PING

Verifica la connessione tra il computer e un dispositivo di rete.

  • ping www.google.it: Verifica la connessione a Internet.

PATHPING

Versione avanzata di ping utile se ci sono più router tra il PC e il dispositivo che si sta testando.

TRACERT

tracert mostra il percorso che un pacchetto fa per arrivare all'indirizzo specificato.

  • Esempio: tracert www.google.it

POWERCFG

Gestione avanzata dell'energia del PC.

  • powercfg /hibernate on e powercfg /hibernate off: Gestione ibernazione.
  • powercfg /energy: Dettagliato rapporto sul consumo di energia.

RECIMG

Comando per creare un'immagine di sistema in Windows 8.

NET USER

Mostra gli utenti registrati nel PC con le rispettive password.

DCOMCNFG

Consente di configurare impostazioni specifiche di DCOM.

MKDIR/MD

Crea una cartella.

MOVE

Sposta una cartella.

RD /s

Elimina una cartella.

Batch e Script VBS

  • Comandi batch come @echo off e script VBS come wshshell sono usati per l'automazione.

Altri Comandi Utili

  • gpedit.msc: Criteri di gruppo.
  • msconfig: Utilità di configurazione di sistema.
  • mstsc: Desktop remoto.
  • regedit: Editor del registro di sistema.
