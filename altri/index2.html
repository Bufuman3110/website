Comandi CMD e Strumenti di Sistema

Rete

  • netsh wlan show profiles - Visualizza le reti memorizzate.
  • driverquery - Elenca i driver del computer.
  • shutdown /r /o - Riavvia il PC e accede alle opzioni avanzate di avvio.
  • sfc /scannow - Controlla e ripara file di sistema.
  • tasklist - Mostra le attività in esecuzione.
  • taskkill /im nome.exe - Termina un processo.
  • ipconfig - Visualizza l'indirizzo IP del computer.
  • ping www.google.it - Controlla la connessione a Internet.
  • tracert indirizzo - Mostra il percorso di un pacchetto.

Gestione File

  • mkdir nome_cartella - Crea una nuova cartella.
  • move - Sposta file o cartelle.
  • rd /s nome_cartella - Elimina una cartella e tutto il suo contenuto.

Utilità e Strumenti

  • eventvwr.msc - Visualizzatore eventi di sistema.
  • regedit - Editor del registro di sistema.
  • msconfig - Utilità di configurazione di sistema.
  • taskmgr - Gestione attività.
